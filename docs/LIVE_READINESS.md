# Live Readiness

CryptoAI is currently paper-only.

Before live trading is considered, require objective evidence:

1. Sustained paper profitability across enough trades.
2. Low drawdown and stable equity curve.
3. Correct accounting reconciliation.
4. Reliable provider health and quote freshness.
5. No duplicate exposure issues.
6. Strategy-level performance report.
7. Live mode must fail closed if data is stale, simulated, or degraded.

AI should advise only. Risk and live safety gates must retain final authority.
